CourseHub [LMS]
==========================================

# Description
  A simple and powerful PHP + MySQL learning management system (LMS) that allows institutions, admins, and students to manage courses, resources, announcements, uploads, and academic information in one centralized platform

# Installation and use
```
$ git clone https://github.com/j-kangethe/CourseHub.git
```
```
$ cd CourseHub
```
```
$ composer install
```

# Setup the database
```
/config/db.php
```
